# This is my first Python project built while learning programming fundamentals.

#So this is a expense tracker which tracks your daily expenses.
# Features-
   1. You can add your expenses inputting its name, category, quantity and its price.
   2. View the expenses you have added according to its category or view it all at once.
   3. It can also show the total number of expenses you have input.
   4. You can delete the expenses u don't want using the name of the expense.
   5. You can also edit the name, category, price and quantity of an existing expense.

# Technologies I have used are mainly-
    -Python 
    -JSON
    
# How to run it-
1. Open project folder in VS Code
2. Run the following command in terminal:
   python expense_tracker.py
